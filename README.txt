Author: Daniel, Reckziegel
Operating System: Ubuntu 18.04.4 LTS
Title: Breakout

_____________________________________________________________________________________________________________________________________________________________________

Spielprinzip:
Es entspricht in etwa einem Solo-Pong, bei dem man mit dem Schläger den Ball so lenken soll, dass der Ball die „Bricks“ trifft, die sich am oberen Bildschirmrand befinden 
und diese zum Verschwinden bringt. Wenn der Schläger, wie bei Pong, den Ball verliert, bekommt man einen neuen Ball, 
verliert aber nach dem Verlust einer bestimmten Anzahl von Bällen das Spiel. Erst wenn alle „Bricks“ abgeräumt wurden, gilt ein Level als gewonnen.

Tastaturbelegung:    		„space“ = Spielstart
				        		„a“     = Paddlebewehung nach links
				        		„d“     = Paddlebewehung nach rechts
				       			 „q“	= Spiel beenden

______________________________________________________________________________________________________________________________________________________________________
Game principle:
It is roughly equivalent to a solo pong, in which one should steer the ball with the racket in such a way that the ball hits the "bricks" that are at the top of the screen and makes them disappear. 
If the racket loses the ball, as in pong, you get a new ball, but you lose the game after losing a certain number of balls. Only when all “bricks” have been cleared is a level considered won.

control: 	"space" = game start
			"A" = paddle movement to the left
			"D" = paddle movement to the right
			"Q" = end game
